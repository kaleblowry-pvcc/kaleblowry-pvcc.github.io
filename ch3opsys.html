Operating Systems Terminology
Terms Definition
Operating system A software that controls the overall operation of a computer, enabling users to store, retrieve files, and execute programs.
Job queue A storage organization in which jobs are ordered in a "first in, first out" (FIFO) fashion.
FIFO "First-in, first-out" order.
Interactive processing A feature in which the user can communicate directly with a program or computer interface through remote terminals.
Time-sharing A design in which operating systems provide services to multiple users simultaneously.
Multiprogramming A time-sharing technique where time is divided into intervals, restricting job execution to one interval at a time.
Application software Programs designed for performing tasks specific to a machine's utilization.
System software Software that performs tasks common to computer systems in general.
Utility software Programs that perform essential tasks in a computer installation but are not included in the operating system.
Graphical user interface (GUI) An interface where objects such as files and programs are represented as icons on the display.
Shell An older interface where users communicate with the system through textual commands.
Kernel The core of an operating system, responsible for managing system resources and communication between hardware and software.
Device driver Software that communicates with hardware peripherals to execute operations.
File manager Manages file access and storage space allocation for creating new files.
Memory manager Allocates memory space for programs.
Virtual memory An imaginary memory space created by swapping data and programs between main memory and mass storage.
Bootstrapping The process of loading the operating system from mass storage into main memory.
ROM Read-Only Memory, a type of non-volatile memory that stores permanent data and instructions for a device to start up.
Deadlock A condition where two or more processes are blocked because each is waiting for a resource held by another process.
Spooling A technique of holding data for output at a later, more convenient time.
Super user (administrator) A user with privileged access to the operating system, allowing them to manage and maintain the system.
Multi-factor authentication A security process requiring users to provide more than one form of identification to log in to an account.
