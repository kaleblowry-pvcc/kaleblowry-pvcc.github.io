Term Definition
Machine language An encoding system where a sequence of instructions are encoded as numeric digits
Debugging The act of locating and correcting errors in programs
Assembler Programs that were developed to convert mnemonic expressions into machine language instructions
Assembly language A mnemonic system for collectively representing programs
Translator A program that was written to translate high-level primitive programs into machine-language programs
Interpreter An alternative to translators, it is another means of implementing third-generation languages
Cross-platform Software Software that is independent of an operating system's and hardware's design, executable throughout a network
Natural language Languages such as English, German, and Latin
Formal language Programming languages
Machine Independence A program that can run on multiple types of computers or computer architectures
High-Level Programming Language A programming language with strong abstraction from computer details
Object-oriented programming (OOP) A paradigm viewing a software system as a collection of objects, each capable of performing actions and interacting
Scripting Language A subset of imperative programming languages used for administrative tasks rather than complex programs
Integer Numeric data consisting of whole numbers
Float Numeric data that might contain non-whole values
String Character data such as a literal constant or variable
Primitive data type A set of basic data types from which others are constructed
Data Structure The conceptual shape or arrangement of data
List Used to store multiple items in a single variable; created using square brackets
Tuple A collection that is ordered and unchangeable; written with round brackets
Set An unordered, unchangeable collection written with curly brackets
Dictionary Stores data values in key-value pairs; ordered, changeable, and without duplicates
Function A set of instructions for performing a task, reusable by other program units
Local Variables Declared within a function, only accessible there
Global Variable Variables not restricted to a particular part of a program
Source Program A program written in a language other than machine code
Object Program A compiled or assembled result of a source program
